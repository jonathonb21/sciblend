# SciBlend quick path

Developed by Jonathon Brunson (jonathonbrunson21@gmail.com).

1. Install ParaView macros from `Paraview Macros/`
2. Export static or animated X3D from ParaView
3. Install the SciBlend folder as a Blender 4.2+ extension
4. Import with axis and scale matching your simulation coordinates

Repository: https://github.com/jonathonb21/sciblend
